⚙️ One-time Setup Required

You need a free Google OAuth Client ID to enable Drive sync. Takes ~3 minutes:

  1. Go to console.cloud.google.com
  2. Create a project → Enable Google Drive API
  3. Go to APIs & Services → Credentials → Create Credentials → OAuth 2.0 Client ID
  4. Choose Web application, add your file's URL to Authorised origins
  5. Copy the Client ID and paste it below, then save the file
